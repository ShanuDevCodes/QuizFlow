# Screenshots

Drop the app screenshots here using these exact filenames so they render in the top-level
`README.md`:

| File | Screen |
|---|---|
| `01-loading.png` | Loading / shimmer skeleton |
| `02-question.png` | Question screen (answering) |
| `03-reveal.png` | Answer revealed (correct + wrong) |
| `04-streak.png` | Streak badge lit (3+ in a row) |
| `05-results.png` | Results screen |
| `06-theme.png` | Light vs. dark theme (or dynamic color) |

PNG or JPG both work; PNG is preferred for UI screenshots. Portrait screenshots look best.
